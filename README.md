# [웹페이지 만들기 with Hugo Template](https://2blackcow.github.io/)

## 설명
- 정적사이트 생성기 Hugo를 활용해서 만든 전북대학교 컴퓨터공학부 이현우의 포트폴리오 사이트입니다.
- 페이지에서 소개, 프로젝트, 경력 등을 확인하실수 있습니다.
